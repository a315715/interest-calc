# interest-calc

利息收入反推計算工具。輸入「月收入 / 本金 / 年利率」其中兩項，自動算出第三項，可同時比較兩個方案（A / B）。

線上網址：https://a315715.github.io/interest-calc/

## 檔案結構

只有一個檔案：`index.html`，純 HTML + CSS + JavaScript，**沒有任何外部套件或框架依賴**，也沒有後端、沒有 API。整個工具的邏輯都在這個檔案裡，直接改這個檔案就好。

## 核心公式（單利概算）

```
年收入   = 月收入 × 12
有效利率 = 年利率 × (1 - 稅率)      // 稅率預設 0，選填
反推本金   = 年收入 ÷ (有效利率 / 100)
反推年利率 = (年收入 ÷ 本金) ÷ (1 - 稅率) × 100
反推月收入 = 本金 × (有效利率 / 100) ÷ 12
```

注意：這是**單利**概算，不是複利。若商品為複利計息，實際數字會有落差。

## 常見維護情境

- **改配色 / 字體大小** → 改 `<style>` 裡 `:root` 的 CSS 變數即可
- **改成複利公式** → 只需改 `<script>` 裡的 `compute()` 函式
- **新增方案 C** → 複製 `state.a` / `state.b` 其中一份，並在 HTML `<body>` 裡加對應的 `<div class="card">`
- **手機輸入時鍵盤跳掉問題** → 目前用 `updateResultOnly()` 只更新結果數字、不重繪整張卡片來避免，若要改輸入欄位結構要注意這點
